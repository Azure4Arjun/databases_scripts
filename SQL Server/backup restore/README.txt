backup all databases.sql - Бекап всех баз данных на сервере в одно место.
backup log script.sql - Бекап лога отдельной БД в файл вида dbname_YYYYMMDD_HHMISS.trn
monitor backup restore database.sql - Скрипт показывает процент выполнения и оставшееся время для операций BACKUP/RESTORE
restore log from more files.sql - Восстановление БД из нескольких лог файлов. Файлы должны быть отсортированны в алфовитном порядке.
